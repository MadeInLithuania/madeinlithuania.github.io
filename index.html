Riceify
The first rice manager.

Get it here.
What's that ?
Riceify is a ricing manager made with C++ and used on Unix distros such as Manjaro, Arch, Void, Ubuntu ...
It is also lightweight, intuitive and easy to install which makes the manager useful.
This manager allows you to switch rices easily and without wasting time by editing manually config files and switching by hand.
How does it work ?
On creation
First, it creates a folder with the rice name in ~/Riceify/Rices, then it copies the home directory to the rice folder with all hidden files.
And, it gets all configurations such as .bashrc, .zsh, i3, polybar and more.
Warning : it doesn't copy visible folders !
On switch
It will ask you to choose a rice, then it copies the rice folder contents, e.g. ~/Riceify/rices/Sample/ to your home directory.
Demonstration of rice switching.
Installation
By source

First, clone this repo :
git clone https://github.com/ZukiLTU/Riceify
Then, go to the created folder :
cd Riceify
Finally, install and run with this command :
make install
